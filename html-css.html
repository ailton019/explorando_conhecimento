Módulo: HTML & CSS

O que é HTML & CSS?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar a estrutura e o esqueleto de páginas web. CSS (Cascading Style Sheets) é a folha de estilo utilizada para definir a aparência visual, o layout e o design dessa estrutura. Juntos, formam a dupla fundamental que constrói tudo o que vemos visualmente na internet.

Aulas Básicas

  1. Estrutura Semântica: O uso correto de tags como <header>, <main>, <section> e <footer>.
  2. Elementos de Conteúdo: Criação de títulos (<h1> a <h6>), parágrafos (<p>), links (<a>) e imagens (<img>).
  3. Seletores e CSS Básico: Como selecionar elementos HTML e aplicar cores, fontes e planos de fundo.
  4. Box Model (Modelo de Caixa): Compreensão de margin (margem), border (borda), padding (espaçamento interno) e content (conteúdo).
  5. Layouts Modernos: Construção de layouts flexíveis e responsivos usando Flexbox e CSS Grid.
  6. Responsividade: Uso de @media queries para adaptar a página a celulares, tablets e desktops.

Curiosidades

  • 📅 O HTML foi criado em 1991 pelo cientista da computação Tim Berners-Lee, o criador da própria World Wide Web (WWW).
  • 🎨 O CSS foi proposto em 1994 por Håkon Wium Lie como uma forma de separar a lógica do estilo visual.
  • 📱 O HTML5 (lançado em 2014) revolucionou a web ao remover a dependência de plugins como Flash para rodar áudio e vídeo.
  • 💡 O CSS moderno é extremamente poderoso, contando com recursos como variáveis nativas, gradientes complexos e animações 3D sem precisar de JavaScript.
  • 🧩 HTML **não** é uma linguagem de programação, mas sim uma linguagem de marcação. Já o CSS é uma linguagem de estilo.
