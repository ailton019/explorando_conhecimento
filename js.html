O que é JavaScript?

JavaScript (JS) é uma linguagem de programação interpretada, leve e orientada a objetos de alto nível. Originalmente concebida para rodar diretamente nos navegadores web e adicionar dinamismo, comportamento e interatividade às páginas HTML, ela hoje se expandiu para rodar em servidores (Node.js), aplicativos móveis e até inteligências artificiais.

Aulas Básicas

  1. Sintaxe e Variáveis: Aprender a declarar variáveis modernas usando let e const (evitando o antigo var).
  2. Tipos de Dados: Manipular textos (Strings), números (Numbers), booleanos (Booleans) e estruturas como Listas (Arrays) e Objetos (Objects).
  3. Funções: Entender a declaração de blocos de execução reaproveitáveis: function somar(a, b) { return a + b; } e Arrow Functions.
  4. Estruturas de Controle: Condicionais (if, else, switch) e laços de repetição (for, while, map, forEach).
  5. Manipulação da DOM: Interagir com o HTML da página, escutando cliques, alterando textos e modificando estilos CSS em tempo real usando document.getElementById().
  6. Programação Assíncrona: Aprender a lidar com dados externos e requisições HTTP usando fetch(), Promises e a sintaxe async/await.

Curiosidades

  • 📅 O JavaScript foi criado em apenas 10 dias por Brendan Eich, em maio de 1995, enquanto trabalhava na Netscape Communications.
  • ☕ O nome "JavaScript" foi uma jogada de marketing para pegar carona no enorme sucesso da linguagem Java da época, embora as duas não tenham nenhuma relação técnica.
  • ⚙️ A especificação oficial da linguagem chama-se ECMAScript (ES), mantida por uma comissão internacional (TC39).
  • 🌍 Todo navegador moderno vem com um motor (engine) interpretador de JavaScript embutido, como o poderoso motor V8 do Google Chrome e Edge.
  • 🚀 Graças ao Node.js (lançado em 2009), desenvolvedores podem programar tanto o Front-End (interface) quanto o Back-End (servidor) usando apenas JavaScript!
