O que é GitHub?

O GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele atua como uma rede social para desenvolvedores e fornece ferramentas de colaboração de ponta, como revisão de código, gerenciamento de projetos, controle de bugs e automação de testes e implantações na nuvem.

Aulas Básicas

  1. Conexão Remota: Criar um repositório no GitHub e vinculá-lo ao Git local usando git remote add origin.
  2. Clonagem: Baixar projetos existentes na nuvem usando git clone [url-do-repositorio].
  3. Sincronização: Enviar alterações locais com git push origin [branch] e baixar alterações remotas com git pull.
  4. Pull Requests (PR): Enviar código modificado para que outros membros da equipe revisem e mesclem na branch principal do projeto.
  5. Issues e Projetos: Organizar o desenvolvimento, documentando tarefas a fazer, bugs descobertos e melhorias desejadas.
  6. GitHub Pages: Hospedar de forma totalmente gratuita páginas estáticas (HTML, CSS e JS) direto de um repositório Git.

Curiosidades

  • 🐙 O mascote do GitHub chama-se Octocat (um híbrido de gato e polvo). Ele foi criado por Simon Oxley, o mesmo designer que criou o passarinho azul do Twitter.
  • 🏢 O GitHub foi comprado pela Microsoft em 2018 por 7,5 bilhões de dólares, uma das maiores aquisições da história da tecnologia.
  • ❄️ O GitHub Arctic Code Vault é uma iniciativa que salvou todos os repositórios públicos ativos em 2020 em filmes digitais especiais guardados a 250 metros de profundidade na Noruega, planejados para resistir a 1.000 anos.
  • 💻 É considerado o maior repositório de código-fonte aberto do mundo, contendo mais de 100 milhões de desenvolvedores e mais de 370 milhões de repositórios.
  • 🤖 A plataforma conta com o GitHub Copilot, uma inteligência artificial que sugere código em tempo real diretamente no seu editor de código.
