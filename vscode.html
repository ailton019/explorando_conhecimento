Visual Studio Code

Módulo: VS Code

O que é o Visual Studio Code?

O Visual Studio Code (ou VS Code) é um editor de código-fonte leve, gratuito e extremamente poderoso desenvolvido pela Microsoft. Compatível com Windows, macOS e Linux, ele possui suporte nativo para JavaScript, TypeScript e Node.js, e conta com um gigantesco ecossistema de extensões que adicionam suporte a praticamente qualquer linguagem de programação existente.

Aulas Básicas

  1. Interface Geral: Navegar pela Barra de Atividades (Activity Bar), Barra Lateral, Editor e Barra de Status.
  2. Command Palette (Paleta de Comandos): O atalho definitivo Ctrl + Shift + P (ou Cmd + Shift + P) para acionar qualquer comando ou configuração no editor.
  3. Loja de Extensões: Instalar complementos essenciais de produtividade como Prettier (formatador), Live Server (servidor local de teste) e temas visuais personalizados.
  4. Atalhos de Teclado: Dominar atalhos como Ctrl + P para abrir arquivos rapidamente, Alt + Up/Down para mover linhas de código e Ctrl + D para selecionar ocorrências duplicadas.
  5. Terminal Integrado: Abrir e executar comandos e scripts diretamente dentro do editor usando o atalho Ctrl + '.
  6. Git Integrado: Controlar e gerenciar commits, branches, diferenças (diffs) e envios de código remoto sem precisar sair do VS Code.

Curiosidades

  • 💻 Lançado em 2015 pela Microsoft, o VS Code é um aplicativo desktop construído em cima do framework **Electron**, o que significa que ele roda sobre tecnologias web: HTML, CSS, JavaScript e Node.js!
  • 📊 Nas pesquisas anuais com desenvolvedores realizadas pelo Stack Overflow, o VS Code é eleito consecutivamente, por larga margem, o ambiente de desenvolvimento de software mais popular do mundo.
  • 🌐 Existe uma versão 100% web do editor. Se você acessar qualquer repositório no GitHub e simplesmente apertar a tecla de ponto final (.) no seu teclado, um VS Code completo abrirá direto no seu navegador.
  • 💡 O núcleo de digitação do VS Code chama-se editor **Monaco**, que a Microsoft desenvolveu originalmente para ferramentas baseadas em navegadores antes do próprio VS Code existir.
  • 🛠️ Embora seja muito poderoso, ele é considerado um "editor de código" e não uma IDE (Ambiente de Desenvolvimento Integrado) completa e pesada, mantendo-se incrivelmente rápido e eficiente no consumo de memória.
