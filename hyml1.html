Módulo: HTML

O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. Ele estrutura o conteúdo que será exibido no navegador.

Aulas Básicas

  1. Estrutura básica: <html>, <head>, <body>
  2. Títulos e parágrafos: <h1> a <h6>, <p>
  3. Links e imagens: <a href>, <img src>
  4. Listas: <ul>, <ol>, <li>
  5. Tabelas: <table>, <tr>, <td>

Curiosidades

  • 📅 O HTML foi criado em 1991 por Tim Berners-Lee.
  • 🌐 A primeira versão tinha apenas 18 tags!
  • 📱 HTML é essencial para páginas responsivas, junto com CSS e JavaScript.
  • 🧩 O HTML5 trouxe muitas melhorias, como suporte a vídeo, áudio e semântica.
  • 💡 HTML **não** é linguagem de programação — é uma linguagem de marcação!
